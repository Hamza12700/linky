## Getting Started

### Prerequisites

- Go - [Get here](https://go.dev/doc/install)

### Installation and Setup

1. **Clone the repository**:
   ```
   git clone https://github.com/wavly/shawty.git
   ```
2. **Navigate to the project directory**:
   ```
   cd shawty
   ```
3. **Run the application**:
   ```
   go run .
   ```
